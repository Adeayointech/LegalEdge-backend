❓ STUCK? COMMON ISSUES:

"Can't find Variables tab"
→ Make sure you clicked the SERVICE box (not just the project)
→ Should see: Overview | Variables | Settings | Deployments | Logs

"New Variable button is grayed out"
→ You might not have permission
→ Ask project owner to add you as collaborator

"It's deploying forever"
→ Wait 5 minutes
→ Check "Logs" tab to see what's happening
→ Check "Deployments" tab to see deployment status

"I made a typo in a variable"
→ Click on the variable to edit it
→ Or delete it (trash icon) and add again
