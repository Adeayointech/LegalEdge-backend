# Law Firm Platform

A comprehensive case management system for law firms, built with modern technology stack.

## Tech Stack

- **Backend**: Node.js + Express + TypeScript
- **Database**: PostgreSQL
- **ORM**: Prisma
- **Frontend**: React + TypeScript
- **Authentication**: JWT + 2FA

## Features (V1)

1. ✅ Authentication & Role-Based Access Control
2. ✅ Case-Centric Management (Firm → Branches → Lawyers → Clients → Cases → Documents)
3. ✅ Smart Document Management with Tagging
4. ✅ Filed/Unfiled Document Tracker
5. ✅ Deadline & Hearing Management with Reminders
6. ✅ Full Audit Trail
7. ✅ Document Version Control
8. ✅ Advanced Search & Filtering
9. ✅ Multi-Branch Firm Support
10. ✅ Analytics Dashboard

## Project Structure

```
/backend          - Express API server
/frontend         - React web application
/prisma           - Database schema and migrations
/shared           - Shared types and utilities
```
